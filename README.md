## 🍄‍🟫 О проекте


Этот проект автоматически собирает и проверяет **бесплатные VPN-серверы** из открытых источников. Обновление происходит **каждые 3 часа** через GitHub Actions, поэтому подписки всегда актуальны.

📊 **Текущая статистика**: 
- ✅ **Рабочих источников**: 25/25
- 🚀 **Всего серверов**: ~4400+
- ⏱ **Обновление**: раз в три часа

  ## 📱 Как использовать

  <details>
  <summary><b>📱 Android — v2rayNG</b></summary>

  ### Первоначальная настройка
  1. Открой v2rayNG
  2. Нажми `+` → **"Импорт подписки из буфера"**
  3. Вставь ссылку на любой файл из таблицы выше

  ### Обновление прокси
  1. Открой v2rayNG
  2. Нажми на три точки (⋮) справа от подписки
  3. Выбери **"Обновить подписку"**
  4. Дождись завершения обновления

  ### Включение прокси
  1. Выбери рабочий сервер из списка (нажми на него)
  2. Нажми кнопку **подключения** внизу экрана (значок V)
  3. Подтверди VPN-соединение, если система попросит
  4. При успешном подключении появится значок ключа в статус-баре

  </details>

  <details>
  <summary><b>📱 iOS — V2Box</b></summary>

  ### Первоначальная настройка
  1. Открой V2Box
  2. Перейди в **"Конфигурации"** → `+` → **"Импортировать V2Ray URL из буфера"**
  3. Вставь ссылку

  ### Обновление прокси
  1. Открой V2Box
  2. Перейди в **"Конфигурации"**
  3. Потяни список вниз для обновления (pull to refresh)
  4. Или нажми на подписку → **"Обновить"**

  ### Включение прокси
  1. Выбери рабочий сервер из списка
  2. Нажми кнопку **"Подключить"** на главном экране
  3. Разреши добавление VPN-конфигурации в настройках iOS
  4. При успешном подключении появится значок VPN в статус-баре

  </details>

  <details>
  <summary><b>💻 Windows — Throne</b></summary>

  ### Первоначальная настройка
  1. Открой Throne
  2. Нажми **"Профили"** → **"Добавить профиль из буфера"**
  3. Вставь ссылку

  ### Обновление прокси
  1. Открой Throne
  2. Перейди в **"Профили"**
  3. Нажми правой кнопкой на подписку → **"Обновить подписку"**
  4. Или используй комбинацию **Ctrl+U** для обновления всех подписок

  ### Включение прокси
  1. Выбери рабочий сервер из списка (двойной клик)
  2. Нажми кнопку **"Подключить"** или клавишу **Enter**
  3. В трее появится уведомление об успешном подключении
  4. Проверь статус подключения в главном окне

  </details>

  <details>
  <summary><b>🐧 Linux — NekoRay</b></summary>

  ### Первоначальная настройка
  1. Открой NekoRay
  2. Нажми **"Программа"** → **"Добавить подписку"**
  3. Вставь ссылку

  ### Обновление прокси
  1. Открой NekoRay
  2. Нажми **"Программа"** → **"Обновить подписки"**
  3. Или используй комбинацию **Ctrl+R**
  4. Дождись завершения обновления всех подписок

  ### Включение прокси
  1. Выбери рабочий сервер из списка (двойной клик)
  2. Или нажми правой кнопкой → **"Запустить"**
  3. Нажми **"Запустить прокси"** в главном окне
  4. Проверь статус подключения — должна появиться скорость трафика

  </details>

  ## 💡 Полезные советы

  - **Регулярно обновляй подписки** — прокси могут перестать работать
  - **Проверяй пинг** перед подключением — выбирай серверы с наименьшей задержкой
  - **Используй тест скорости** встроенный в приложение для проверки работоспособности
  - **Добавь несколько подписок** из таблицы для резервных вариантов

  Изменения:
  - Разделил инструкции на три части: первоначальная настройка, обновление и включение
  - Добавил конкретные шаги для каждого действия
  - Указал горячие клавиши где применимо
  - Добавил раздел с полезными советами в конце
